# ThriftyChef docs (GitHub Pages)

Enable GitHub Pages for this repository:

1. GitHub repo → **Settings** → **Pages**
2. Source: **Deploy from a branch**
3. Branch: `Thrifty-chef-implementations` (or `main`) / folder: `/docs`
4. Save

Privacy Policy URL for App Store Connect:

```
https://njdbsprojects20093736.github.io/FridgeWise/privacy-policy.html
```

If Pages is served from `main`, merge/copy `docs/privacy-policy.html` into `main` as well.
